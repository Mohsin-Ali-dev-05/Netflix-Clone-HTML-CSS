# Netflix Clone Landing Page

A Netflix-inspired landing page created using HTML5 and CSS3.

## Project Overview

This project replicates the user interface of Netflix's landing page for educational and learning purposes. The focus was on improving frontend development skills, responsive design techniques, CSS layouts, and modern web page structuring.

## Features

- Responsive Landing Page
- Hero Section
- TV Showcase Section
- Download Offline Section
- Watch Everywhere Section
- Kids Profile Section
- FAQ Section
- Footer Navigation
- Embedded Videos
- Mobile Responsive Design

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries

## Learning Outcomes

Through this project I learned:

- Website layout structuring
- Flexbox positioning
- Responsive web design
- Background overlays
- CSS transitions and hover effects
- Video embedding
- Professional landing page design
